**VENDO** 

aplikasi manajemen bisnis vending machine berbasis web yang memungkinkan pengguna untuk mengelola operasional vending machine secara terpusat, mulai dari pengelolaan produk, stok, transaksi, hingga pengguna.

Aplikasi ini dibangun menggunakan: : 
⚡ Vite + React (Frontend), 
🔗 Nest js (Backend), 
🗄️ PostgreSQL (Supabase), 
🎞️ Tailwind (framework CSS),
💵 Midtrans (Payment Gateway),
📡 MQTT (Komunikasi dengan vending machine)

🚀 Fitur Utama : 
🔐 Autentikasi pengguna (Login), 
🍎 manajemen Produk, 
📰 manajemen inventaris (stock, vending machine), 
🧑 Manajemen User, 
📈 Statistik penjualan, 
💵 Transaksi terintegrasi (Midtrans)

🏗️ Arsitektur Sistem 

                                Frontend (Vite + React)
                                           ↓
      Midtrans (Payment gateway) <- Nest JS (Backend) -> MQTT (Vending Machine)
                                           ↓
                              Supabase (Database PostgreSQL)
